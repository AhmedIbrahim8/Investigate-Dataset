Introduction¶

Dataset Description¶

This data set contains information about 10,000 movies collected from The Movie Database (TMDb), including user ratings and revenue.

Certain columns, like ‘cast’ and ‘genres’, contain multiple values separated by pipe (|) characters. There are some odd characters in the ‘cast’ column. Don’t worry about cleaning them. You can leave them as is. The final two columns ending with “_adj” show the budget and revenue of the associated movie in terms of 2010 dollars, accounting for inflation over time.

  1. Which Movie has the higher revenue and its genre ?
  2. Which Movie has the lowest revenue and its genre ?
  3. Which Movie has the higher budget?
  4. Which Movie has the higher Profit and its genre?
  5. Which Movie has the lowest Profit and its genre?
  6. Which Director has the highest revenue?
  7. Which Director has the highest profit?
  8. Relation between revenue and budget ?

def splitted(column):
    # split the data in the column by | to be converted to list for better handling
    df_new_data[column]=df_new_data[column].apply(lambda x: x.split("|"))
def generate_conclusion(column):
    # We need dictionaries for :
    # 1- counts for each element of the data
    # 2- sum of the revenue for each element of the data
    # 3- sum of the popularity for each element of the data
    # 4- sum of the profit for each element of the data
    dictionary=dict()
    dictionary_revenue=dict()
    dictionary_popularity=dict()
    dictionary_profit=dict()
    # i is an index for the row to access the row by indexing
    i=0
    for a in df_new_data[column]:
        for b in a:
            # Here, We count of the data
            dictionary[b]=dictionary.get(b,0)+1
            # Here, We sum the revenue of each typeof data
            dictionary_revenue[b]=dictionary_revenue.get(b,0)+df_new_data['revenue'].iloc[i]
            # Here, We sum the popularity of each typeof data
            dictionary_popularity[b]=dictionary_popularity.get(b,0)+df_new_data['popularity'].iloc[i]
            # Here, We sum the profit of each typeof data
            dictionary_profit[b]=dictionary_profit.get(b,0)+df_new_data['profit'].iloc[i]
        # increment the i for the next row
        i=i+1
    # Generate a data frame for organizing our conclusion
    df=pd.DataFrame(dictionary,index=['counts'])
    df=df.append(pd.DataFrame(dictionary_revenue,index=['total_revenue']))
    df=df.append(pd.DataFrame(dictionary_popularity,index=['total_popularity']))
    df=df.append(pd.DataFrame(dictionary_profit,index=['total_profit']))
    # Transpose the data frame to better visualization
    df=df.T
    # Number of counts differs from  each type of data so, we calculated the mean for the revenue and profit and popularity
    df['mean_popularity']=df['total_popularity']/df['counts']
    df['mean_revenue']=df['total_revenue']/df['counts']
    df['mean_profit']=df['total_profit']/df['counts']
    return df
